# João Vitor Bruschi — Portfolio (Astro)

This is a personal portfolio built with Astro + Tailwind.

## Quickstart

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the dev server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build && npm run preview
   ```
